# What is this ?
This application shows you the vulnerabilities recorded in the database.
# Which Database ?
[National Vulnerability Database](https://nvd.nist.gov/)
# Features
 - Search Bar 
 - File Save Feature
 - Severity System
 - A small info screen at the bottom to learn more about the Vulnerability
